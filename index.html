Connect to Google Sheets

1
Go to console.cloud.google.com → create or select a project
2
Search Google Sheets API → Enable it. Also enable Google Drive API
3
Go to APIs & Services → Credentials → + Create Credentials → OAuth 2.0 Client ID → type: Web application
4
Add your GitHub Pages URL to Authorised JavaScript origins e.g. https://yourusername.github.io
5
Copy your Client ID (ends in .apps.googleusercontent.com) and paste below
